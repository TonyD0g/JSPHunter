# JSPHunter [Java 8]

基于污点分析和模拟栈帧技术的JSP Webshell检测    

`仅做学习记录`,**95%**代码来源于JSPFinder和JSPKiller   

尝试对其进行优化检测逻辑和重构,以作为自己学习污点分析和模拟栈帧技术的检验和毕业设计.
